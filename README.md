# Flutter Calculator App

A simple calculator application built using Flutter.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Clean and simple UI

## Technologies Used

* Flutter
* Dart

## Getting Started

1. Clone the repository

git clone https://github.com/ShubhaVishwakarma/calculator.git

2. Open the project

cd calculator

3. Run the app

flutter run
